# 君臣佐使推断 + 剂量古今换算 (v0.3)

`herb_hermes/formula_analysis/` 在 v0.2 方剂谱系之上，对方剂组成做两件事：把
历代剂量折算为现代克数，并推断每味药的君臣佐使配伍角色。

## 剂量古今换算 (`dosage.py`)

### 数字与单位解析
- `cn_num` 解析中文/阿拉伯数字：`三`→3、`十二`→12、`七十`→70、`一百`→100、
  `半`→0.5，兼容 `二兩半`（→2.5）。
- `parse_dose_token` 从「药名（…）」括注中抽取首个剂量：`三兩`、`半斤`、
  `十二枚`、`各三錢`（识别 `各` 共享量）。

### 朝代换算因子（克 / 兩）
| 朝代 | 1兩≈ | 体系 |
| --- | --- | --- |
| 漢·魏晉·南北朝 | 15.6 g | 1斤=16兩, 1兩=24銖（柯雪帆/上海中医药考） |
| 隋·唐 | 41.3 g | 大制（1斤≈661g） |
| 宋·金·元 | 40.0 g | 1兩=10錢=100分 |
| 明 | 37.0 g | |
| 清 | 37.3 g | |
| 民國（旧市制） | 31.25 g | 16兩=500g |
| 現代（新市制） | 50.0 g | 10兩=500g |

- 容量（漢制经方）：1升≈200ml、1合≈20ml，按 ml 报告。
- 计数单位（枚/個/片/條/握/束/杯/盞…）不定量，原样报告；`一字`≈0.65g（估值）。
- 古今折算存学派分歧（尤其经方一两=15.6g vs 临床折算），结果统一附 `note`，
  **仅供科研参考，不作处方依据**。

折算示例（系统自动）：
```text
桂枝湯（宋本，1兩≈40g）：桂枝60g 芍藥60g 生薑60g 甘草40g 大棗6枚（全方≈220g）
麻黃湯（1兩≈37g）：麻黃55.5g 桂枝37g 甘草18.5g 杏仁50個
```

## 君臣佐使推断 (`roles.py`)

`infer_roles` 综合四类信号给出角色 + 依据 + 置信度（含剂量=高，仅位置=中）：

1. **剂量权重（主）**：折算克数越大权重越高；计数单位（如杏仁五十個）不计入权重。
2. **方名命名**：方名含某药（桂枝湯→桂枝、麻黃湯→麻黃）→ 该药优先为 **君**。
3. **功能词典**：`甘草`→**使**（调和诸药）；`生薑/大棗`→**佐**（调和营卫）；
   `桔梗/牛膝/升麻/柴胡`等且非君 →**使**（引经/载药）。
4. **位置**：剂量缺失或并列时以原方位次定先后。

赋值规则：君（最高权重或命名药）→ 臣（次高、≥0.5×君量，至多 2 味）→
佐（其余兼证/佐制）→ 使（调和/引经）。输出按 君→臣→佐→使 排序。

经典验证（系统输出）：
```text
桂枝湯：君桂枝 · 臣芍藥 · 佐生薑/大棗 · 使甘草      （与教材一致）
麻黃湯：君麻黃 · 臣桂枝 · 佐杏仁 · 使甘草            （与教材一致）
四君子湯：君人參 · 臣白朮 · 臣茯苓 · 使甘草
```

> 这是**透明的启发式推断**，每条都给依据与置信度，便于研究者复核；不同医家对
> 君臣佐使本就存在解读差异。古籍单一刻本剂量缺漏时（如某刻本六味地黃丸缺熟地黃
> 用量），系统会在「代表版本组」中优选剂量最完整的刻本，仍可能受原文影响。

## 接口

- CLI：`python -m herb_hermes.cli formula 桂枝湯`（谱系 + 君臣佐使 + 剂量）
- API：`GET /formula/{name}`（含 `analysis`）或 `GET /analyze/{name}`
- 前端「方剂谱系」页：君臣佐使表 + 剂量柱状图（按角色着色）
